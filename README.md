# Interview AI Web App Documentation

## Overview

Interview AI is a web application that allows users to practice interviews with AI.
The system generates interview questions, evaluates answers, and provides feedback and scores.

## Tech Stack

- Next.js (Fullstack)
- TypeScript
- MongoDB
- Mongoose
- TailwindCSS
- Zustand
- Recharts
- AI API (Gemini)

## Main Features

- Authentication
- AI Interview Chat
- AI Feedback & Score
- Dashboard
- Interview History
- Reports
- User Profile

## System Architecture

```
Client → Next.js API Routes → MongoDB
                            → AI API
```
